Sobre o Projeto

Projeto Integrador

Visme

Comunicação organização

O aplicativo Visme se encontra na área de comunicação e criação de conteúdo, onde suas funções são consideradas fáceis de compreender mas que necessita de uma certa curva de aprendizado para utilizar todos os seus recursos. Nele é possível criar apresentações mais elaboradas conforme suas necessidades.

Prós:

  • Além de você conseguir criar seus próprios slides e templates, é possível utilizar templates gratuitos criados pelo próprio site
  • Existem imagens gratuitas para utilização
  • É possível trazer seus conteúdos para dentro do aplicativo, sendo eles: Imagens, Templates, Vídeos, gráficos
  • Existem objetos animados para utilização dentro do aplicativo
  • Fácil de criar seus slides
  • Grande variação de conteúdo para utilização
  • Visualmente bonito para o espectador

    • Contras:

  • Para ter um acesso mais completo ao aplicativo, o usuário deve melhorar seu pacote
  • Não é possível utilizar este aplicativo sem acesso a internet
  • O aplicativo não permite edições em outros aplicativos que não sejam o Visme
  • Existem alguns bug’s visuais devido ao aplicativo ser relativamente novo
